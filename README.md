# Portfolio Vision

Professional business portfolio website with retro-inspired design aesthetics.

## Project Status

✅ **Frontend Foundation Complete**
- React + TypeScript + Vite scaffold
- Retro design system (muted earth tones, serif headings)
- Core components (Button, Card, Navigation, Footer)
- Home page with hero, stats, services preview, and featured work
- Routing configured for all main pages
- Mobile-responsive navigation

🚧 **In Progress**
- Services detail pages
- Work portfolio with case studies
- About page with team section
- Contact form with validation

## Quick Start

```powershell
# Navigate to frontend
cd frontend

# Install dependencies (if not already done)
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` to see the site.

## Design System

**Color Palette** (Retro/Vintage):
- Cream (#f5f1e8) - Primary background
- Sand (#e8dcc4) - Secondary background
- Rust (#a0522d) - Primary accent
- Terracotta (#c47d5f) - Hover states
- Olive (#6b705c) - Text secondary
- Charcoal (#2b2d2f) - Dark sections

**Typography**:
- Headings: Crimson Pro (serif)
- Body: Inter (sans-serif)
- Code/Mono: IBM Plex Mono
